# Alea 品牌规范

## 六项核心 Token

```css
:root {
  --bg: oklch(95.85% 0.0098 87.47);
  --surface: oklch(98.45% 0.0067 97.35);
  --fg: oklch(27.15% 0.0094 67.42);
  --muted: oklch(50.64% 0.0191 79.25);
  --border: oklch(90.76% 0.0157 86.43);
  --accent: oklch(59.87% 0.1327 41.67);
}
```

这些值分别由 PRD 指定的 `#F4F1EA`、`#FBFAF5`、`#2A2622`、`#6B6459`、`#E5E0D5`、`#C0613B` 转换而来。

## 字体栈

- 展示：`"Source Serif 4", "Noto Serif SC", Georgia, serif`
- 正文与控件：`"Spline Sans", "Noto Sans SC", system-ui, sans-serif`
- 数据与编号：`"Spline Sans Mono", ui-monospace, Menlo, monospace`

## 视觉姿态

1. 以米纸画布、暖白面板、细描边与极浅环形阴影建立可信的纸面档案感。
2. 陶土色只用于主行动、选中态和一个最高信号品牌点，避免促销化。
3. 绿、红、赭金只表达命中、失败、过期等语义，并同时配文字或图标。
4. 大比分、赔率、时间、金额、版本号和审计编号统一使用等宽数字。
5. 动效克制且可降级；无霓虹、赌场金、闪烁、金币、红包或紧迫性诱导。
